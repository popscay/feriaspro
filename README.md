# VacaFlow — Employee Vacation Management System

A full-stack employee vacation management application built with:

- Java 17
- Spring Boot
- React
- PostgreSQL
- Docker Compose

The application manages employees and vacation requests with role-based permissions.

---

## How to Run

### Requirements

Before running the project, make sure you have installed:

- Docker
- Docker Compose

You do **not** need to install PostgreSQL locally. PostgreSQL runs inside Docker.

---

## Run with Docker

From the project root folder, run:

```bash
docker compose up --build
